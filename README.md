> A curated list of awesome resources, tools, frameworks, platforms, and research papers for Edge Computing.

Edge Computing is the practice of processing data closer to where it is generated to reduce latency, improve performance, and increase privacy. This list is intended for developers, engineers, researchers, and anyone interested in edge computing.

## General Resources

- [Edge Computing (Wikipedia)](https://en.wikipedia.org/wiki/Edge_computing) – Overview of edge computing concepts.
- [Edge Computing - Gartner IT Glossary](https://www.gartner.com/en/information-technology/glossary/edge-computing) – Business-focused definition and insights.
- [Edge Computing World](https://www.edgecomputingworld.com/) – Industry event focused on edge computing.

## Frameworks & SDKs

- [EdgeX Foundry](https://www.edgexfoundry.org/) – Open-source edge computing platform managed by LF Edge.
- [Azure IoT Edge](https://azure.microsoft.com/en-us/services/iot-edge/) – Microsoft’s platform for deploying cloud intelligence locally.
- [AWS IoT Greengrass](https://aws.amazon.com/greengrass/) – Extends AWS to edge devices for local execution.
- [Open Horizon](https://www.lfedge.org/projects/horizon/) – Secure and autonomous management of edge devices and apps.
- [KubeEdge](https://kubeedge.io/en/) – Extends Kubernetes capabilities to edge environments.

## Platforms & Services

- [Google Distributed Cloud Edge](https://cloud.google.com/distributed-cloud/edge) – Google’s fully managed edge solution.
- [IBM Edge Application Manager](https://www.ibm.com/products/edge-application-manager) – Manages workloads on edge devices at scale.
- [Cloudflare Workers](https://workers.cloudflare.com/) – Serverless functions running at the network edge.
- [Fastly Compute@Edge](https://www.fastly.com/products/compute-at-edge) – Edge cloud platform for running apps closer to users.
- [Verizon 5G Edge](https://www.verizon.com/business/products/5g/edge-computing/) – Low-latency edge computing services over 5G.

## Edge AI & Machine Learning

- [NVIDIA Jetson](https://developer.nvidia.com/embedded-computing) – AI edge devices and SDKs.
- [OpenVINO Toolkit](https://www.intel.com/content/www/us/en/developer/tools/openvino-toolkit/overview.html) – Intel toolkit for optimized deep learning at the edge.
- [TensorFlow Lite](https://www.tensorflow.org/lite) – Lightweight ML models for mobile and edge devices.
- [PyTorch Mobile](https://pytorch.org/mobile/home/) – Edge deployment for PyTorch models.

## Networking & 5G

- [Open Networking Foundation (ONF)](https://opennetworking.org/) – Open source networking and edge platforms.
- [5G-ACIA](https://5g-acia.org/) – Alliance for 5G in industrial applications.
- [Akraino Edge Stack](https://www.lfedge.org/projects/akraino/) – Open-source edge cloud project for 5G, AI, and IoT.

## IoT & Edge Devices

- [Balena](https://www.balena.io/) – Deployment and management platform for IoT devices.
- [ThingsBoard](https://thingsboard.io/) – Open-source IoT platform with edge capabilities.
- [Home Assistant](https://www.home-assistant.io/) – Edge-controlled home automation platform.
- [Edge Impulse](https://www.edgeimpulse.com/) – Machine learning development platform for edge devices.

## Security & Privacy

- [Confidential Computing Consortium](https://confidentialcomputing.io/) – Promoting secure computation at the edge.
- [Zero Trust Edge (Gartner)](https://www.gartner.com/en/documents/4006636) – Gartner’s concept for securing edge computing environments.
- [Edge Computing Security (Cloud Security Alliance)](https://cloudsecurityalliance.org/research/working-groups/edge-computing/) – Security guidelines and best practices.

## Open Source Projects

- [LF Edge Projects](https://www.lfedge.org/projects/) – Multiple open-source projects under Linux Foundation’s edge computing umbrella.
- [OpenYurt](https://openyurt.io/) – Extends Kubernetes to edge computing scenarios.
- [Fledge](https://www.lfedge.org/projects/fledge/) – Open-source industrial edge platform.
- [KubeStellar Console](https://github.com/kubestellar/console) – Open source AI-powered multi-cluster Kubernetes dashboard for managing hybrid edge and cloud environments. Supports real-time observability with 20+ CNCF integrations and AI-guided cluster operations. CNCF Sandbox project (Apache 2.0).
